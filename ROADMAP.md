# 🗺️ Slot Machine Project Roadmap

## Phase 1: Foundation (초기 기반 구축)
- [ ] 프로젝트 멀티모듈 구조 설계 (`:cmp:core`, `:cmp:feature:slot`, etc.)
- [ ] 'The Calm Ink' 공통 테마 및 디자인 시스템 기초 구현
- [ ] Koin 의존성 주입 및 네비게이션 설정

## Phase 2: Core Interaction (핵심 인터랙션)
- [ ] MVI 기반 Slot Machine State 설계 (Idle, Spinning, Result)
- [ ] 레버(Lever) 인터랙션 및 드래그 로직 구현
- [ ] 3개 릴(Reel)의 무작위 회전 및 정지 로직 (Logic Only)

## Phase 3: Visual & UX Polish (시각화 및 고도화)
- [ ] 릴 회전 애니메이션 고도화 (Blur 효과 및 속도감)
- [ ] 당첨 결과에 따른 시각적 피드백 (Jenny's Micro-interaction)
- [ ] 햅틱 피드백 및 효과음 적용

## Phase 4: Data & Release (데이터 및 배포)
- [ ] 간단한 로컬 저장소(Room/DataStore) 연동 (최고 점수 저장 등)
- [ ] 릴리즈 빌드 최적화 및 배포 준비
